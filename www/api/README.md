# API 

## Структура запроса

- URL запроса: **/api/index.php** 
- Тип запроса: **POST**
- Вид запроса: строка **JSON**, переданная в *POST*-переменной **request**

## Методы

### auth. Авторизация.
### points.  Получение истории баллов
### transaction. Начисление/списание баллов
