# API 

## Структура запроса

- URL запроса: **/api/index.php** 
- Тип запроса: **POST**
- Вид запроса: строка **JSON**, переданная в *POST*-переменной **request**

## Методы

### auth. Авторизация.
#### Запрос

```
{
    "method":"auth",
    "args":{
        "login":"79277374026",
        "password":"*************"
    }
}
 ```

#### Ответ

```
{
    "errorCode":0,
    "errorMessage":"",
    "result":{
        "session_id":"********************"
    }
}
```

*session_id* после этого используется в других методах для авторизации запроса
